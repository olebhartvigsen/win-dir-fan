# FanFolder v1.1.5

Date: 2026-04-20

Changelog since v1.1.4:
- Fan menu now opens reliably on rapid clicks — no more silent reopens without animation.
- Much faster menu open: icons are prewarmed in the background and reused across opens.
- Taskbar icon, Alt+Tab, live preview and tray now all show the same unified app icon.
- File icons with cloud/OneDrive overlays no longer render upside down or with a black background.
